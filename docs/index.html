プライバシーポリシー

1. 入力データの取り扱い

本アプリで入力された家計情報（収入、支出、貯金残高、メモなど）は、家計管理の目的のみに使用します。これらのデータは、お使いの端末（ブラウザ）の中にのみ保存され、サーバーには送信されません。

2. 入力しないでいただきたい情報

クレジットカード番号、有効期限、セキュリティコード、カード裏面番号、暗証番号、カード名義人、銀行口座番号、ログインID、パスワードなどの重要情報は、本アプリでは入力しないでください。家計管理のため、金額のみを記録してください。

3. データの削除

「設定」ページからいつでもご自身のデータをすべて削除できます。

4. 第三者提供

個人情報を第三者に販売・提供することはありません。

5. 広告・アフィリエイトについて

本アプリの一部には、広告・アフィリエイトリンクが含まれる場合があります。該当箇所には「PR」または「※本ページには広告・PRが含まれる場合があります。」と表示します。

6. お問い合わせ

お問い合わせ先は、運営者によって後から設定されます。
